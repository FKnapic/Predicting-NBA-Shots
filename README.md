# Predicting NBA Shots

Machine learning project to predict NBA shot success using player and game features. This project explores various algorithms including logistic regression, SVM, Naive Bayes, neural networks, random forests, and boosting, with full data preprocessing, analysis, and evaluation.

## Project Overview

The goal of this project is to predict whether an NBA shot attempt will be successful based on features such as player distance from the basket, closest defender distance, game time, shot type, and more.  

This project is inspired by Brett Meehan's Stanford CS229 final project on NBA shot prediction. It demonstrates how different machine learning methods can be applied to a real-world sports analytics problem and compares their performance.  

## Dataset

The dataset comes from [Kaggle NBA Shot Data 2014-2015](https://www.kaggle.com/datasets) and contains 122,502 examples of shot attempts.  

Features include:

- Player and team information (player id, player name, home/away)
- Game information (period, game clock, shot clock, game outcome)
- Shot information (shot distance, points type, shot number)
- Defensive information (closest defender, distance to closest defender)

> **Note:** For privacy and licensing reasons, the raw dataset is not included.
